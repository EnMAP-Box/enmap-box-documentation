Usage:

1. Start the algorithm from the Processing Toolbox panel.

2. Select the vector layer containing your categorised polygons and the raster layer serving as your target grid, then click :guilabel:`run`.

    .. figure:: source/usr_section/usr_manual/processing_algorithms_includes/classification/img/classfraction_interface.png
       :align: center

3. View the result in the map viewer. Each class is represented as a separate band.

    .. figure:: source/usr_section/usr_manual/processing_algorithms_includes/classification/img/class_fraction_result.png
       :align: center
