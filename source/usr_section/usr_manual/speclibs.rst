Spectral Libraries
##################

Spectral Libraries are collections of (i) spectral profiles and (ii) attributes that described the
profile data.

Spectral Library Model
----------------------

The EnMAP-Box realized spectral libraries using vector layers. This allows to overcome
a couple of disadvantages associated with traditional file-based spectral library formats.
